# SMUTE HOTEL

Smute Hotel is a responsive hotel website template.

## Template features

- Mobile First strategy
- HTML5 & CSS3 responsive theme
- Clean design & layout
- SEO Markup
- Responsive Images
- Testimonial Carousel
- 4 Pages
- Well-commented Code
- Easy Setup
